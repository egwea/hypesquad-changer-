# 🛡️ Discord HypeSquad House Changer & Remover

An automated Python script to easily change or completely remove your Discord HypeSquad House badge using direct Discord API requests. Lightweight, clean, and open-source.

سكربت بايثون خفيف لتغيير شارة بيت الهايسبكواد (HypeSquad) في ديسكورد أو حذفها نهائياً بكل سهولة عبر طلبات API مباشرة.

---

## ✨ Features / المميزات

* **Fast Execution:** Changes or deletes the badge in less than a second.
* **No Client Needed:** Works directly via Discord API endpoints.
* **Safe & Clean:** 100% open-source code with no external hidden dependencies.
* **سرعة التنفيذ:** يغير أو يحذف الشارة في أقل من ثانية.
* **بدون تطبيق:** يعمل مباشرة عبر الاتصال بـ API ديسكورد.
* **آمن ونظيف:** كود مفتوح المصدر بالكامل ولا يحتوي على أي ملغمات أو برمجيات خفية.

---

## 🛠️ How to Use / طريقة الاستخدام

### 1. Prerequisites / المتطلبات
Make sure you have Python installed, then install the `requests` library:
تأكد من تثبيت بايثون على جهازك، ثم ثبت مكتبة الطلبات عبر الأمر التالي:
```bash
pip install requests
2. Get Your Token / استخراج التوكن الخاص بك
Open Discord in your browser, press F12 to open Developer Tools.

Go to Application -> Local Storage -> https://discord.com.

Search for token and copy your token value.

⚠️ WARNING: Never share your token with anyone!

افتح ديسكورد من المتصفح، واضغط F12 لفتح أدوات المطورين.

توجه إلى Application ثم Local Storage ثم رابط ديسكورد.

ابحث عن كلمة token وانسخ القيمة الخاصة بك.

⚠️ تحذير: لا تشارك التوكن الخاص بك مع أي أحد نهائياً!

3. Run the Script / تشغيل السكربت
Paste your token inside the TOKEN variable in the script.

Run the script and choose your option!

ضع التوكن الخاص بك داخل متغير TOKEN في الكود.

شغل السكربت واختر الخيار الذي يناسبك!
